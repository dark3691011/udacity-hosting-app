# Pipeline

## Build

1. Docker: create image docker
2. Steps:

- Node/install: install node version 14.15
- Install front-end
- Install backend
- Run lint front-end
- Build front-end
- Build backend

## Deploy

1. Docker: create image docker
2. Steps:

- Node/install: install node version 14.15
- Eb/setup: setup Elastic Beanstalk
- Aws-cli/setup: setup aws-cli
- Checkout
- Deploy both apps
